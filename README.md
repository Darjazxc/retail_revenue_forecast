# Прогноз выручки американской ритейл-компании | Revenue Forecasting for Аmerican Retail Company

> Комплексный анализ данных крупной американской ритейл-компании с применением методов машинного обучения и статистического моделирования для прогнозирования выручки.



## Описание проекта

Данный проект представляет собой детальный анализ данных компании с целью прогнозирования выручки на основе исторических показателей продаж, поведения клиентов и операционных метрик.

### Основные задачи:
- **Исследовательский анализ данных (EDA)** — изучение структуры и качества данных
- **Анализ продаж** — выявление трендов и сезонности
- **Сегментация клиентов** — кластерный анализ для группировки покупателей
- **Корреляционный анализ** — определение факторов, влияющих на выручку
- **Прогнозирование** — построение модели для предсказания будущей выручки

---

## Структура данных

Проект работает с шестью таблицами:
- **Orders** — информация о заказах
- **Products** — характеристики товаров
- **Order Address** — географические данные
- **Customers** — профили клиентов
- **Returns** — данные о возвратах
- **Ship Mode** — способы доставки

### Ключевые переменные:
- `Sales` — объем продаж
- `Profit` — прибыль
- `Quantity` — количество товара
- `Discount` — предоставленные скидки
- `Category` / `Sub-Category` — категории товаров
- `Region` / `State` / `City` — географическое распределение
- `Segment` — сегмент клиентов

## Технологии и инструменты

```python
- Python 3.9
- Pandas — обработка и анализ данных
- NumPy — численные вычисления
- Matplotlib / Seaborn — визуализация
- Scikit-learn — машинное обучение (KMeans)

```

---

## Основные этапы анализа

### 1. Предобработка данных
- Загрузка и объединение таблиц
- Обработка пропущенных значений
- Преобразование типов данных (даты, категории)
- Создание производных признаков

### 2. Разведочный анализ
- Статистический анализ распределений
- Временные ряды продаж
- Географический анализ
- Анализ по категориям товаров

### 3. Кластерный анализ
- **K-Means кластеризация** — сегментация клиентов/заказов
- **Иерархическая кластеризация** — построение дендрограмм
- Анализ характеристик кластеров

### 4. Моделирование
- Корреляционный анализ факторов
- Построение регрессионной модели (OLS)
- Оценка значимости переменных

---

## Основные выводы

- **Положительная динамика**: Показатели компании демонстрируют устойчивый рост год к году

- **Три кластера клиентов**: Выявлены различные группы покупателей с разными паттернами поведения

- **Прогноз на 2 года**: Ожидается продолжение роста выручки на основе анализа трендов

- **Ключевые факторы**: Определены основные драйверы выручки для бизнес-стратегии
